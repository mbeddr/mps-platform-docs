    MPS and Dependency Versions Overview¶

    There are many different MPS versions and builds available. Each version is build on top of a certain IDEA Platform ⧉ version with a certain Java version. Jetbrains uses their own Java runtime which is bundled since MPS 2019.2: The Jetbrains Java Runtime (JBR) ⧉ (except for the generic package which is shipped without the JBR).

    When you setup a Gradle based MPS build you can also chose to download the corresponding MPS and JBR versions ⧉ for the build process, making you build independent of the system it is build on. For this you need to know which MPS version was build based on which JBR version to obtain matching versions.

    The following table provides an exhaustive overview over build meta-data of all MPS versions for all platforms (generic/linux/mac/win). The knowledge was extracted directly from the artifacts provided on the MPS download page ⧉, namely the files build.number, build.properties, and jbr/release.
